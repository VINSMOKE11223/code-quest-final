# 🚀 CodeQuest – Gamified Learning Platform

CodeQuest is a **gamified coding learning platform** designed to make programming education interactive, engaging, and rewarding. The platform uses game mechanics like **avatars, quizzes, rewards, and leaderboards** to motivate learners.

This project was built as part of a **hackathon / student project** to demonstrate how learning platforms can be improved using gamification.

---

# 🌟 Features

### 🔐 User Authentication

* Secure login system
* Password encryption using **bcrypt**
* Authentication using **JWT tokens**

### 🧑 Avatar System

Users can create and customize their avatars while learning.

### 📚 Learning Dashboard

* Interactive dashboard for learners
* Track progress and learning paths

### 🧩 Quizzes & Challenges

* Chapter-based quizzes
* Coding challenges
* Skill-based progression

### 🏆 Rewards & Gamification

* Earn rewards for completing tasks
* Motivation through game elements

### 🎨 Interactive UI

* Custom animations
* Interactive cursor effects
* Smooth user experience

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```
git clone https://github.com/YOUR_USERNAME/code-quest-final.git
```

```
cd code-quest-final
```

---

## 2️⃣ Install Backend Dependencies

```
cd backend
npm install
```

---

## 3️⃣ Run the Server

```
npm run dev
```

or

```
npm start
```

Server will run on:

```
http://localhost:3000
```

---

## 4️⃣ Run Frontend

Open:

```
frontend/login.html
```

in your browser.
